## 4. Decisoes principais

- Fonte secundaria nao sustenta regra normativa sozinha.
- SempreJU SF4 foi classificada como `THIRD_PARTY_TECHNICAL_CATALOG` e `SECONDARY_TECHNICAL`.
- SX3 confirmado do ambiente prevalece para tipo, tamanho, dominio e validacao de campos Protheus.
- XML sintetico nao e fonte normativa.
- Claims exigem escopo, status, evidencia e revisao quando verificadas.
