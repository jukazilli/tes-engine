# Escopo do MVP

Status: ACCEPTED

## Incluido no MVP

- Fundacao tecnica do monorepo.
- SPA Angular com PO UI.
- API e worker NestJS.
- Infraestrutura local.
- Fronteiras Nx.
- Quality gates e CI.
- Governanca documental.
- Fluxo conceitual de upload, processamento, cenarios, revisao, aprovacao e exportacao.

## Fora do MVP neste momento

- ORM e migrations.
- Autenticacao.
- Entidades persistidas.
- Parser XML.
- BullMQ.
- Motor fiscal.
- Wizard funcional.
- Exportacao real.
- Gravacao direta na SF4.

## Evolucoes futuras

- Multi-tenant fisico.
- Importacao de dados Protheus.
- Similaridade de TES.
- Layout MILE inicial.
- Golden files fiscais.
- Threat model completo.
