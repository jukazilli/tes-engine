# Gestao de mudancas

Status: ACCEPTED

## Fluxo

1. Definir escopo fechado.
2. Conferir fonte de verdade.
3. Implementar apenas o corte aprovado.
4. Atualizar documentos canonicos.
5. Registrar relatorio de execucao.
6. Executar validacoes.
7. Criar commit convencional.

## Regras

- Nao ocultar validacoes falhas.
- Nao misturar refatoracoes esteticas com mudanca de comportamento.
- Nao antecipar decisoes registradas como pendentes.
- Nao mover documentos historicos sem beneficio claro.
- Nao incluir arquivos fora do projeto.
