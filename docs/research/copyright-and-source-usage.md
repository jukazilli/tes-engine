# Copyright e uso de fontes

Status: ACCEPTED

## Politica

O repositorio deve guardar resumos, localizadores, IDs e decisoes. Nao deve guardar copia integral
de paginas, tabelas, PDFs, XSDs ou catalogos de terceiros.

## Permitido

- Link para fonte.
- Pequeno resumo em palavras proprias.
- Localizador como secao, titulo, nome de arquivo ou campo isolado.
- Trecho curto quando necessario e juridicamente aceitavel.

## Nao permitido

- Copiar catalogo SF4 inteiro.
- Copiar paginas TDN/TOTVS completas.
- Versionar XSDs oficiais sem decisao explicita de licenca e necessidade.
- Usar XML fiscal real de cliente como evidencia publica no repositorio.
