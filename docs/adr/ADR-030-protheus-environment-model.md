# ADR-030: Modelo de ambiente Protheus

Status: ACCEPTED

Data: 2026-06-25

## Decisao

Ambiente Protheus pertence a uma filial e guarda somente metadados nao secretos: tipo, versoes,
release e codigos Protheus como texto.

## Consequencias

Nao ha conexao com Protheus, credenciais, URLs internas ou teste de conectividade neste corte.
